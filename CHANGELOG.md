# Changelog

All notable changes to SatTrack will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Development branch workflow
- Automatic tag creation on merge to main
- GitHub Actions workflow for releases

## [1.0.4] - 2024-12-XX

### Added
- PWA install functionality
- Install button component
- Proper 192x192 and 512x512 icons
- Auto-tagging workflow for releases

### Changed
- Removed unnecessary debug console logs
- Improved PWA manifest configuration

## [1.0.3] - Previous Release

### Added
- Initial release features
- Satellite tracking
- Pass predictions
- Transponder data

## [1.0.2] - Previous Release

### Fixed
- Various bug fixes

## [1.0.1] - Previous Release

### Fixed
- Initial bug fixes

## [1.0.0] - Initial Release

### Added
- Real-time satellite tracking
- Pass prediction system
- Transponder frequency data
- PWA support
- Offline functionality
- Compass mode
- Sound alerts
